Go 插件

Posted @ 2018-09-09 · 140 Words · Tags: Go · Hits:

Go提供了一种插件机制，可用于在运行时动态链接其他Go代码。在Go1.8中，仅支持Linux。

定义和使用插件

package main

import "fmt"

var V int

func F() { fmt.Printf("Hello, number %d\n", V) }

使用以下命令build:

$ go build -buildmode=plugin

在你的程序中加载并使用:

p, err := plugin.Open("plugin_name.so")
if err != nil {
    panic(err)
}

v, err := p.Lookup("V")
if err != nil {
    panic(err)
}

f, err := p.Lookup("F")
if err != nil {
    panic(err)
}

*v.(*int) = 7
f.(func())() // 打印 "Hello, number 7"
