Go 判断

Posted @ 2018-09-09 · 992 Words · Tags: Go · Hits:

if

使用if进行基本的条件判断, 基本使用示例:

a := 5
b := 6
if a == b {
    fmt.Print("a is equal to b\n")
} else {
    fmt.Print("a is not equal to b\n")
}

a is not equal to b

func returnValues(ok bool) (int, bool) {
    return 5, ok
}

func main() {
    a := 5
    b := 5
    if a == b {
        fmt.Print("a is equal to b\n")
    } else {
        fmt.Print("a is not equal to b\n")
    }

    if n, ok := returnValues(true); ok {
        fmt.Printf("ok is true, n: %d\n", n)
    } else {
        fmt.Printf("ok is false, n: %d\n", n)
    }
}

a is equal to b
ok is true, n: 5

在if语句中声明的变量的作用域是该语句。 以下示例无法编译通过:

func returnValues(ok bool) (int, bool) {
    return 5, ok
}

func main() {
    if n, ok := returnValues(true); ok {
        fmt.Print("ok is true, n: %d\n", n)
    } else {
        fmt.Print("ok is false, n: %d\n", n)
    }

    // n is only visible within if statement, so this fails compilation
    fmt.Printf("n: %d\n", n)
}

./main.go:19:24: undefined: n
exit status 2

switch

switch基本用法:

stmt := "if"
switch stmt {
case "if", "for":
    fmt.Printf("stmt ('%s') is either 'if' or 'for'\n", stmt)
case "else":
    fmt.Printf("stmt is 'else'\n")
default:
    fmt.Printf("stmt is '%s'\n", stmt)
}

stmt (‘if’) is either ‘if’ or ‘for’

注意，与C++不同，case语句不会涉及到下一个case，你不必在每个case的末尾都放置break。

switch也可以用来判断类型.

a := 1
switch a {
case 1, 3:
    fmt.Printf("a is 1 or 3\n")
case 2:
    fmt.Printf("a is 2\n")
default:
    fmt.Printf("default: a is %d\n", a)
}

a is 1 or 3

注意一个case可以处理多个值(上例中为1和3)。

下通

与大多数其他语言(例如C ++或Java)不同，你无需使用break即可停止继续执行以下case。这种默认行为通常是错误的来源。

Go中可以通过以下方式达到这种效果:

a := 1
switch a {
case 1:
    fmt.Printf("case 1\n")
    fallthrough
case 2:
    fmt.Printf("caes 2\n")
}

case 1
caes 2

switch判断strings

我们可以对字符串进行switch判断:

s := "foo"
switch s {
case "foo":
    fmt.Printf("s is 'foo'\n")
case "bar":
    fmt.Printf("s is 'bar'\n")
}

s is ‘foo’

空switch表达式

switch表达式可以为空，在这种情况下，case是布尔表达式，而不是常量:

func check(n int) {
    switch {
    case n > 0 && n%3 == 0:
        fmt.Printf("n is %d, divisible by 3\n", n)
    case n >= 4:
        fmt.Printf("n is %d (>= 4)\n", n)
    default:
        fmt.Printf("default: n is %d\n", n)
    }
}

check(3)
check(4)
check(6)
check(1)

n is 3, divisible by 3 n is 4 (>= 4) n is 6, divisible by 3 default: n is 1

case判断顺序

如果多个case都为true，会发生什么？

在上面的示例中 6匹配n> 0 && n％3 == 0和n≥4表达式。

如上所见,只会执行最先满足的情况

assignment in switch

switch n := rand.Intn(9); n {
case 1, 2, 3:
    fmt.Printf("case 1, 2, 3: n is %d\n", n)
case 4, 5:
    fmt.Printf("case 4, 5: n is %d\n", n)
default:
    fmt.Printf("default: n is %d\n", n)
}

default: n is 6

类型switch

如果switch中是一个接口，则可以根据接口的底层值类型进行switch:

func printType(iv interface{}) {
    // inside case statements, v is of type matching case type
    switch v := iv.(type) {
    case int:
        fmt.Printf("'%d' is of type int\n", v)
    case string:
        fmt.Printf("'%s' is of type string\n", v)
    case float64:
        fmt.Printf("'%f' is of type float64\n", v)
    default:
        fmt.Printf("We don't support type '%T'\n", v)
    }
}

func main() {
    printType("5")
    printType(4)
    printType(true)
}

‘5’ is of type string
‘4’ is of type int
We don’t support type ‘bool’

goto语句

goto语句将控制权转移到同一函数中带有相应标签的语句。

func printIsOdd(n int) {
    if n%2 == 1 {
        goto isOdd
    }
    fmt.Printf("%d is even\n", n)
    return

isOdd:
    fmt.Printf("%d is odd\n", n)
}

printIsOdd(5)
printIsOdd(18)

5 is odd
18 is even

goto不能跳过变量声明:

    goto end
    a := 3
    fmt.Printf("a: %d\n", a)
end:

./main.go:9:7: goto end jumps over declaration of a at ./main.go:10:4
exit status 2
