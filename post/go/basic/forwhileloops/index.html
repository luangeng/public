Go 循环

Posted @ 2019-09-09 · 1059 Words · Tags: Go · Hits: 0

Go中只有for循环,for循环的完整形式如下:

for i := 0; i < 5; i += 2 {
	fmt.Printf("i: %d\n", i)
}

i: 0
i: 2
i: 4

for循环中的每个部分都不是必须的

比如没有初始化:

i := 0
for ; i < 5; i += 2 {
	fmt.Printf("i: %d\n", i)
}

i: 0
i: 2
i: 4

没有判断部分, 循环内部使用break来结束循环

i := 0
for ; ; i += 2 {
	fmt.Printf("i: %d\n", i)
	if i >= 5 {
		break
	}
}

i: 0
i: 2
i: 4
i: 6

没有自增语句

for i := 0; i < 5; {
	fmt.Printf("i: %d\n", i)
	i += 2
}

i: 0
i: 2
i: 4

无限循环

i := 0
for {
	fmt.Printf("i: %d\n", i)
	i += 2
	if i >= 5 {
		break
	}
}

i: 0
i: 2
i: 4

使用range在slice上循环

a := []int{1, 3, 5}
for idx, value := range a {
	fmt.Printf("idx: %d, value: %d\n", idx, value)
}

idx: 0, value: 1
idx: 1, value: 3
idx: 2, value: 5

for循环的其他形式

单个变量for循环:

for i := 0; i < 10; i++ {
    fmt.Print(i, " ")
}

两个或更多变量循环:

for i, j := 0, 0; i < 5 && j < 10; i, j = i+1, j+2 {
    fmt.Println(i, j)
}

这个例子是count初始为0的无限循环

for i := 0; ; {
    fmt.Print(i, " ")
    if i == 9 {
        break
    }
    i++
}

当仅使用测试表达式时（类似于典型的while循环）：

i := 0
for i < 10 {
    fmt.Print(i, " ")
    i++
}

只有自增表达式:

i := 0
for ; ; i++ {
    fmt.Print(i, " ")
    if i == 9 {
        break
    }
}

使用索引和值迭代range：

ary := [5]int{0, 1, 2, 3, 4}
for index, value := range ary {
    fmt.Println("ary[", index, "] =", value)
}

只使用索引迭代

for index := range ary {
    fmt.Println("ary[", index, "] =", ary[index])
}

或者

for index, _ := range ary {
    fmt.Println("ary[", index, "] =", ary[index])
}

只使用value迭代

for _, value := range ary {
    fmt.Print(value, " ")
}

迭代map Iterate over a range using key and value for map (may not be in order):

mp := map[string]int{“One”: 1, “Two”: 2, “Three”: 3} for key, value := range mp { fmt.Println(“map[", key, “] =”, value) } Iterate over a range using just key for map (may be not in order):

for key := range mp { fmt.Print(key, " “) //One Two Three } Iterate over a range using just key for map (may be not in order):

for key, _ := range mp { fmt.Print(key, " “) //One Two Three } Iterate over a range using just value for map (may be not in order):

for _, value := range mp { fmt.Print(value, " “) //2 3 1 } Iterate over a range for channels (exits if the channel is closed):

ch := make(chan int, 10) for i := 0; i < 10; i++ { ch <- i } close(ch)

for i := range ch { fmt.Print(i, " “) }

迭代string（提供Unicode代码点）：

Iterate over a range for string (gives Unicode code points):

utf8str := "B = \u00b5H" //B = µH
for _, r := range utf8str {
    fmt.Print(r, " ") //66 32 61 32 181 72
}
fmt.Println()
for _, v := range []byte(utf8str) {
    fmt.Print(v, " ") //66 32 61 32 194 181 72
}
fmt.Println(len(utf8str)) //7

as you see utf8str has 6 runes (Unicode code points) and 7 bytes. 如您所见，utf8str有6个符文（Unicode代码点）和7个字节。

break 和 continue

break用于提前跳出for循环

i := 0
for {
	i++
	if i > 2 {
		break
	}
	fmt.Printf("i: %d\n", i)
}

i: 1
i: 2

continue用于进行下次循环

for i := 0; i < 3; i++ {
	if i == 1 {
		continue
	}
	fmt.Printf("i: %d\n", i)
}

i: 0
i: 2

在switch中使用Break/continue

package main

import "fmt"

func main() {
	j := 100

loop:
	for j < 110 {
		j++

		switch j % 3 {
		case 0:
			continue loop
		case 1:
			break loop
		}

		fmt.Println("Var : ", j)
	}
}

Var : 101

while循环

Go中没有while循环,可以使用for来实现;比如下面的C++ while循环代码:

int n = 0;
while (n < 3) {
    printf("n: %d\n", n);
    n++;
}

可以写为for循环:

n := 0
for n < 3 {
	fmt.Printf("n: %d\n", n)
	n++
}

n: 0
n: 1
n: 2

更简洁的版本：

for n := 0; n < 3; n++ {
	fmt.Printf("n: %d\n", n)
}

n: 0
n: 1
n: 2
