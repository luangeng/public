Go 日志

Posted @ 2019-09-09 · 407 Words · Tags: Go · Hits:

日志记录是一个非常深入的主题，因为不同的程序具有不同的日志记录要求。

使用fmt.Printf 和 fmt.Fprintf打印日志

日志的一种简单方式是打印在标准输出中(stdout):

fmt.Printf("Logging to %s\n", "stdout")

打印日志到stderr:

fmt.Fprintf(os.Stderr, "Logging to %s\n", "stderr")

使用log包打印日志

标准log包提供了更多的功能:

log.Printf("Logging")
log.Printf("Second line\n")

2019/11/06 02:17:58 Logging
2019/11/06 02:17:58 Second line

与fmt.Printf相比，log.Printf默认情况下会将日志记录到stderr(os.Stderr)并将当前时间添加到每个日志行中… 如果未明确指定，则通过添加\n来确保每个日志单独在一行

打印致命错误:

f, err := os.Open("file.txt")
if err != nil {
    log.Fatalf("os.Open('file.txt') failed with '%s'\n", err)
}

log.Fatalf打印错误日志并且调用os.Exit(1)来终止进程

日志记录到文件

log包可以改变日志输出到哪里,比如文件:

logfile, err := os.OpenFile("test.log", os.O_RDWR|os.O_CREATE|os.O_APPEND, 0666)
if err != nil {
    log.Fatalf("os.OpenFile() failed with '%s\n", err)
}
defer logfile.Close()

log.SetOutput(logfile)
log.Println("Log entry")

日志记录到syslog

当程序运行在Unix时, 日志可以记录到syslog:

syslogger, err := syslog.New(syslog.LOG_INFO, "syslog_example")
if err != nil {
    log.Fatalln(err)
}

log.SetOutput(syslogger)
log.Println("Log entry")
