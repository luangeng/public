Go range

Posted @ 2018-09-09 · 1352 Words · Tags: Go · Hits:

遍历字符串

s := "Hey 世界"
for i, r := range s {
    fmt.Printf("idx: %d, rune: %d\n", i, r)
}

idx: 0, rune: 72
idx: 1, rune: 101
idx: 2, rune: 121
idx: 3, rune: 32
idx: 4, rune: 19990
idx: 7, rune: 30028

请注意，范围假定字符串是UTF-8编码的，并且遍历Unicode字符(rune)而不是字符串的字节。

返回的第一个值i是字符串中符文的字节索引，而不是Unicode字符索引。

要遍历字符，请使用字节索引：

s := "a 世"
for i := 0; i < len(s); i++ {
    c := s[i]
    fmt.Printf("idx: %d, byte: %d\n", i, c)
}

idx: 0, byte: 97
idx: 1, byte: 32
idx: 2, byte: 228
idx: 3, byte: 184
idx: 4, byte: 150

迭代slice

a := []int{3, 15, 8}
for i, el := range a {
    fmt.Printf("idx: %d, element: %d\n", i, el)
}

idx: 0, element: 3
idx: 1, element: 15
idx: 2, element: 8

迭代map

m := map[string]int{
    "three": 3,
    "five":  5,
}
for key, value := range m {
    fmt.Printf("key: %s, value: %d\n", key, value)
}

key: three, value: 3
key: five, value: 5

对string使用range

遍历字节

可以遍历字符串中的字节：

s := "a 世"
for i := range s {
    b := s[i]
    fmt.Printf("idx: %d, byte: %d\n", i, b)
}

idx: 0, byte: 97
idx: 1, byte: 32
idx: 2, byte: 228

遍历runes

遍历字符串中的字符:

s := "Hey 世界"
for i, rune := range s {
    fmt.Printf("idx: %d, rune: %d\n", i, rune)
}

idx: 0, rune: 72
idx: 1, rune: 101
idx: 2, rune: 121
idx: 3, rune: 32
idx: 4, rune: 19990
idx: 7, rune: 30028

在Go语言中，字符串是不可变的字节序列, 参考只读的[]byte切片。

每个字节的范围是0到255。

世界上有非常多的文字.

Unicode标准为每个已知字符定义唯一值。 Unicode称它们为代码点，它们是可以容纳32位的整数。

为了表示Unicode代码点，Go具有rune类型。它是int32的别名。

Go源代码中的文字字符串是UTF-8编码的。

每个Unicode代码点都可以使用1到4个字节进行编码。

在这种迭代形式中，Go假定字符串是UTF-8编码的。 range将每个代码点解码为UTF-8，以字符串形式返回解码的rune及其字节索引。

你会看到最后一个代码点的字节索引跳了3个，因为它之前的代码点代表一个汉字，并且需要3个字节的UTF-8编码。

字符串和UTF-8 Go中string是字节片。可以将任意二进制数据放入其中。

如何解释字节取决于你的代码。

大多数情况下，字符串以UTF-8编码表示Unicode字符串，但在Go源代码中的字符串文字之外，Go不会检查或确保字符串数据形成有效的UTF-8序列。

话虽如此，Go提供了用于处理UTF-8编码数据的功能。

range的行为就是一个例子。

对slice使用range

在slice上使用range有3种变体:

同时获取索引和元素:

a := []int{3, 15, 8}
for i, el := range a {
    fmt.Printf("idx: %d, element: %d\n", i, el)
}

idx: 0, element: 3
idx: 1, element: 15
idx: 2, element: 8

仅获取索引:

a := []int{3, 15, 8}
for i := range a {
    fmt.Printf("idx: %d\n", i)
}

idx: 0
idx: 1
idx: 2 仅获取元素

a := []int{3, 15, 8}
for _, el := range a {
    fmt.Printf("element: %d\n", el)
}

element: 3
element: 15
element: 8

对map使用range

map上的迭代具有不确定的顺序。 实际上，Go编译器会麻烦地将顺序随机化，这样程序员就不必依赖偶然的顺序了。

在map上使用range迭代有3种变体。

同时获取key和value:

m := map[string]int{
    "three": 3,
    "five":  5,
}
for key, value := range m {
    fmt.Printf("key: %s, value: %d\n", key, value)
}

key: five, value: 5
key: three, value: 3

仅获取key:

m := map[string]int{
    "three": 3,
    "five":  5,
}
for key := range m {
    fmt.Printf("key: %s\n", key)
}

key: three
key: five

仅获取value:

m := map[string]int{
    "three": 3,
    "five":  5,
}
for _, value := range m {
    fmt.Printf("value: %d\n", value)
}

value: 3
value: 5

对channel使用range

func fillAndCloseChannel(ch chan int) {
    for i := 0; i < 3; i++ {
        ch <- i + 3
    }
    close(ch)
}

ch := make(chan int)
go fillAndCloseChannel(ch)

for v := range ch {
    fmt.Printf("v: %d\n", v)
}

v: 3
v: 4
v: 5

channel上的range接收通过channel发送的元素。

channel关闭时，迭代结束。

以下是更方便的方法：

ch := make(chan int)
go fillAndCloseChannel(ch)

for {
    v, ok := <-ch
    if !ok {
        break
    }
    fmt.Printf("v: %d\n", v)
}
