ubuntu 16.04安装redis-server

Posted @ 2017-09-07 · 330 Words · Tags: redis · Hits:

ubuntu 16.04安装redis-server

apt-get install redis-server

启动、停止、重启、重新加载、查看状态

/etc/init.d/redis-server start

/etc/init.d/redis-server stop

/etc/init.d/redis-server restart

/etc/init.d/redis-server force-reload

/etc/init.d/redis-server status

进入redis命令行

daemon启动程式，后面加&表示后台启动，后面加配置文件path表示使用该配置文件启动

查看redis是否在运行:

1.检测后台进程是否存在
ps -ef |grep redis

2.检测6379端口是否在监听
netstat -lntp | grep 6379

3.使用redis-cli查看

设置redis密码

vi /etc/redis/redis.conf

后面为设置的密码

requirepass password

打开redis远程访问

用vi打开Redis服务器的配置文件redis.conf

vi /etc/redis/redis.conf

#注释掉下面的一行 #bind 127.0.0.1

修改后，重启Redis服务器，使用密码访问命令行

sudo /etc/init.d/redis-server restart redis-cli -a password

在远程的一台Mac访问Redis服务器

redis配置文件

可参考 http://www.cnblogs.com/kreo/p/4423362.html

redis执行初始化脚本

windows

type data.txt | redis-cli

Linux

cat data.txt | redis-cli
