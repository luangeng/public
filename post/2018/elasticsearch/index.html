elasticsearch

下载elasticsearch， https://www.elastic.co/downloads/elasticsearch 解压

修改配置文件config/elasticsearch.yml 增加 http.cors.enabled: true http.cors.allow-origin: “*” http.cors.allow-headers: Authorization,Content-Type

进入bin 双击elasticsearch.bat，启动

下载elasticsearch-head-master git://github.com/mobz/elasticsearch-head.git 先装node.js 解压，进入目录，npm install； npm run start，然后访问 http://localhost:9100/
