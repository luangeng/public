Linux常用命令

Posted @ 2017-09-08 · 446 Words · Tags: linux · Hits: 0

man 查询命令帮助

info 查询命令文档

whatis 查询命令简介

whereis 查询命令文件位置

which 查询命令运行位置

finger 显示用户的信息

who 显示当前登录用户

whoami 显示用户ID

sudo 以管理员身份运行

passwd 修改密码

alias 显示/设置别名

env 显示/设置环境变量

echo “xxxx” 显示字符串

pwd 显示当前路径

cd - 返回之前的路径

cd .. 返回上层路径

touch xx.txt 新建文件

mkdir xx 新建文件夹

rm 删除文件/夹

rm -rf 强制删除非空文件夹 r递归 f强制

cat 查看文件 -n 显示行号

ls [-al]显示当前目录下结构 a现实隐藏文件/夹 ， l 显示详情

cp 拷贝

mv 剪切

mv name1 name2 重命名文件/夹

mv /a /b/c 移动并重命名文件/夹

tar -zxvf XX.tar.gz 解压缩

find 指定命令 指定文件 ##查找文件 如 find / -name ‘a*.txt’

locate 查找文件

； 顺序执行多个命令

| 管道

> 重定向 覆盖

>> 重定向 追加

grep 查找字符

uname -a 显示系统信息

free 显示内存信息

ifconfig 显示网络信息

route 显示路由信息

netstat -tap 显示网络连接状态

host DNS查询

ps -ef 显示所有进程

df －hl 显示磁盘信息

zip 压缩

unzip 解压缩

shutdown -h now 立即关机

shutdown -r now 立即重启

按键： tab 命令补全

ctr＋C 取消执行

free命令用来查看系统内存状况：

uptime 命令查看系统启动时间

lscpu 查看系统cpu信息

lsblk 查看硬盘和分区

fdisk 硬盘详细信息
